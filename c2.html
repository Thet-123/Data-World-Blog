Programming လောကမှာတင်မဟုတ်ဘူး၊ ဘယ်နေရာမဆို လေ့လာသင်ယူတဲ့နေရာမှာ consistency ကောင်းဖို့လိုပါတယ်။ မြန်မာမှုပြုရရင် ဇွဲကောင်းဖို့လိုပါတယ်။

ဘာကြောင့် consistency ကောင်းဖို့လိုတာလဲပေါ့၊ အကြောင်းရင်းကတော့ consistency ကောင်းကောင်းမထားနိုင်သမျှ ကောက်ရိုးမီးပဲဖြစ်နေပြီး ဘယ်အရာမှ ထွန်းထွန်းပေါက်ပေါက်ဖြစ်လာမှာမဟုတ်ဘူး။ ဒါတင်မဟုတ်ဘူး၊ ဇွဲလုံ့လ ဆုတ်ယုတ်လာပြီဆို လုပ်နေတဲ့ အလုပ်ပေါ်မှာ focus လုပ်ရတာတွေ အားနည်းလာမယ်၊ သည်းခံနိုင်မှုအပိုင်းကလဲ ကျလာလိမ့်မယ်။
ဆိုတော့ consistency ကောင်းအောင် ဘယ်လိုအလုပ်ရမလဲပေါ့။

အပျင်းမကြီးပါနဲ့။
မှားဖတ်မိတာမဟုတ်ဘူး။ အရေးကြီးဆုံးက အဲ့ဒါပဲ၊ ကိုယ်လုပ်စရာရှိတာတွေ၊ ဘာလုပ်ရမယ်ဆိုတာက ကိုယ့်ခေါင်းထဲမှာ ရှိပြီးသား၊ ပွင့်ပွင့်လင်းလင်းပြောရရင် အားလုံးဆီမှာ ကိုယ်ပိုင်အနာဂတ်ဆိုတာတောင် ပုံဖော်ထားကြပြီးသား။ လိုအပ်တာ တစ်ခုတည်းက အားသွန်ခိုက်စိုက် ဇွဲကောင်းကောင်းနဲ့ ကြိုးစားလိုက်ဖို့ပဲ။ ဒါကြောင့် အပျင်းမကြီးပါနဲ့၊ ဒီနေ့ လုပ်စရာရှိတာကို မနက်ဖြန်မရွေ့ဘဲ ဒီနေ့ပြီးအောင် လုပ်ကြည့်ပါ။

Career ဆိုတာက လျှောက်ရမှာ အရှည်ကြီး အဲ့လောက်ကြီးတွေးပြီး စိတ်ဓါတ်ကျစရာမလိုဘူး။ အပိုင်းလေးတွေခွဲချလိုက်၊ ဥပမာ လက်ရှိမှာ PHP လေ့လာနေတယ်၊ ဒါကို ကျွမ်းကျွမ်းကျင်ကျင် သုံးတတ်တဲ့အထိ ဇွဲကောင်းကောင်းနဲ့ လုပ်ပြီး ဖျက်ချပလိုက်၊ ပြီးရင်နောက်တစ်ခုဆက်သွား။ အဲ့လိုမဟုတ်ဘူး ဒီလို task သေးသေးကိုတောင် ဇွဲကောင်းကောင်းနဲ့ မဖြိုချနိုင်ရင် နောက်ရှေ့ဆက်သွားရမယ့် career journey မှာ တိုင်ပတ်သွားမှာပဲ၊ ရောက်သင့်သလောက်လဲ ရောက်မှာမဟုတ်တော့ဘူး။

တစ်ချို့တွေကျတော့ လည်း IT career လစာကောင်းတယ်ဟေ့၊ programmer တွေလစာကောင်းတယ်ဟေ့ဆိုပြီး ဝင်တိုးလာကျ HTML CSS လေးနည်းနည်းပဲရေးရသေးတယ်၊ မလုပ်ချင်ကြတော့ဘူး၊ ဘယ်အလုပ်ဖြစ်ဖြစ် ပေးဆပ်မှုတစ်ခုခုလိုပါတယ်၊ အရေးကြီးဆုံးကတော့ consistency ပေါ့၊ သူနဲ့အတူ အချိန်တွေ၊ တစ်ခြားရင်းနှီးမြုပ်နှံမှုတွေပါလုပ်ရမှာကိုး။ ပိုကောင်းတဲ့ target ဆို ပိုပြီးတော့ ဇွဲကောင်းကောင်းနဲ့ကြိုးစားရလေပဲ။ programming အလုပ်ဆိုတာလည်း တော်ရုံ ဝါသနာမပါတဲ့လူဆို coding မရေးချင်ကြတော့ လုပ်မယ်ဆုံးဖြတ်ထားပြီးရင် ပုံမှန်လူတွေထက် ဇွဲပိုကောင်းမှ တော်ရာကျမှာပါ။

ကိုယ့်မှာ consistency level အားကောင်းလာပြီဆို ဘယ်တွေကောင်းလာမလဲဆိုတာ ကျနော် ကိုယ်တွေ့နဲ့ sharing လုပ်ပေးလိုက်မယ်၊ motivation လည်းရအောင်။ ဇွဲလုံ့လ အားကောင်းလာပြီးဆို သူနဲ့အတူ လိုက်လာမယ့် ကောင်းတဲ့ အရာတွေအများကြီးပဲ။

ကိုယ်လုပ်ရတဲ့ အလုပ်အပေါ် အာရုံစိုက်နိုင်မှု အားကောင်းလာမယ်၊ Focus ကောင်းလာမယ်။
Self-control အားကောင်းလာမယ်။ ဒီနေ့ ဒီ task ကို push လုပ်မယ်၊ ပြီးအောင်လုပ်မယ် စသည်ဖြင့် မိမိကိုယ်ကို ထိန်းချုပ်ပြီး လုပ်နိုင်စွမ်းအားတွေကောင်းလာလိမ့်မယ်။
Focus နဲ့ self-control ကောင်းလာပြီဆိုတာနဲ့ productivity ကလဲအချိုးတူ တတ်လာလိမ့်မယ်၊ career progression တစ်ဖြည်းဖြည်းမြင့်လာမယ်။
Productivity အားကောင်းလာတာနဲ့ I can do it, I did it ဆိုတဲ့ feeling ကိုရလာပြီး မိမိ ကိုယ်ကို confidence ပိုရှိလာမယ်။
ကုန်ကုန်ပြောရရင် ကြိုးစားတဲ့ လူအဖို့ ရမယ့် ဆုလဒ်တွေက စောင့်နေမှာပဲ။
ဒီစာကိုရေးလိုက်ပေမယ့် ကျနော်ကိုယ်တိုင်လည်း consistency အရမ်းကောင်းနေတဲ့လူမဟုတ်၊ ပျင်းတဲ့ အချိန်တွေရှိတယ်။ ဒါပေမယ့် ပြီးရမယ့် အလုပ်ကို အရင် ပြီးအောင် လုပ်တယ်၊ ပြီးမှ ကိုယ်လုပ်ချင်တာကို လုပ်လိုက်တယ်။ ဒီအတွက်လည်း မိမိ progress တွေတိုင်းထားပြီး consistent လုပ်နိုင်အောင် schedule လေးတွေချထားသင့်တယ်၊ ပြီးတော့မှ ကိုယ်ချထားတဲ့ schedule ပေါ်မှာ keep on track လုပ်ပေါ့။ အချိန်အကြောင်းဆက်ပြောရင် time management လုပ်ရတာတွေပါ ပါလာမှာဆိုတော့ ဒီ ဆောင်းပါးမှာကတော့ ဒီလောက်နဲ့ပဲ ရပ်လိုက်ပါတယ်။ ပေးချင်တဲ့ message လေးတွေရမယ်လို့ မျှော်လင့်ပါတယ်။ အရေးအကြီးဆုံးကတော့ တစ်ခုခုလုပ်မယ်၊ချမယ် ဆုံးဖြတ်ထားပြီးရင် ဆုံးခန်းတိုင်အောင် ဇွဲရှိရှိနဲ့လုပ်ပါ၊ ကောက်ရိုးမီး မဖြစ်ပါစေနဲ့။ ဒါပါပဲ။
