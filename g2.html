Browser Extensions that I Use

Category : General

ကိုယ်ကဘယ် profession area မှာပဲအလုပ် လုပ်လုပ်၊ productivity ကောင်းဖို့အတွက်ကိုယ်နဲ့သက်ဆိုင်တဲ့ tools တွေကိုကောင်းကောင်းအသုံးချနိုင်မယ်ဆိုရင် ပိုပြီးအဆင်ပြေချောမွေ့စေပါတယ်။ ဒီနေ့တော့ ကျနော်နေ့စဉ်မှာအသုံးများဖြစ်တဲ့ browser extensions တွေရေးသွားပါမယ်။ တစ်ခြားသုံးကောင်းတာတွေရှိရင်လည်း comment မှာ recommend လုပ်ပေးသွားနိုင်ပါတယ်။

Myanmar Font Tools

ဒီ extension ကတော့တော်တော်များများသိပြီးကြသားပါ။ Zawgyi နဲ့ Unicode ကအခုထိ balance မဖြစ်သေးဘူး၊ တစ်ချို့က zawgyi သုံးနေတုန်း တစ်ချို့ကလည်း Unicode နဲ့၊ ဒီ extension သွင်းထားမယ်ဆို font setting တွေပြောင်းစရာမလိုဘဲ နှစ်မျိုးလုံးနဲ့ compatible ဖြစ်အောင်လုပ်ဆောင်ပေးနိုင်ပါတယ်။ တစ်ခါတစ်လေတော့ Unicode နဲ့ရေးထားတဲ့ website တွေရဲ့ Myanmar font design ကို effect ဖြစ်စေနိုင်ပါတယ်။

https://chrome.google.com/webstore/detail/myanmar-font-tools/ckhifbinlmakgeidlbbmplikmcofaedf

Momentum
ဒီ extension က browser မှာ new tab ဖွင့်တဲ့အချိန်ကိုယ် setting လုပ်ထားတဲ့ info တွေအကုန်ပြန်ပြပေးပါတယ်။ todo list, personal name, weather info အစရှိတာတွေသုံးနိုင်ပါတယ်။
အကြိုက်ဆုံးတစ်ခုကတော့ နေ့တိုင်းမှာမတူညီတဲ့ background photo နဲ့ motivation quote လေးတွေပြပေးတာပဲဖြစ်ပါတယ်။
https://microsoftedge.microsoft.com/addons/detail/momentum/jdoanlopeanabgejgmdncljhkdplcfed

Daily.dev
အခုနောက်ပိုင်း momentum ကိုဖြုတ်ပြီးတော့ daily.dev ဆိုတာကိုသုံးဖြစ်တယ်။ သူကကျ new tab ဖွင့်လိုက်တာနဲ့ development နဲ့ပတ်သတ်တဲ့ article တွေပြပေးတယ်။ ကိုယ့်စိတ်ကြိုက်ရအောင် personalize လုပ်လို့လည်းရတယ်၊ ဥပမာ api, server, security , devops စတဲ့ ကိုယ်ကြိုက်တဲ့ category တွေရွေးပေးထားလို့ရတယ်၊ စာဖတ်ဖြစ်အောင် goal လေးတွေလည်းသတ်မှတ်ထားလို့ရတယ်။ ဒီ extension သွင်းပြီးနောက်ပိုင်း စာပိုဖတ်ဖြစ်လာတယ်။ https://daily.dev/

BlockSite

တစ်ခါတစ်လေ ကိုယ်အလွန်အကျွန်သုံးမိတဲ့ website တွေရှိတယ်၊ ဥပမာ facebook, youtube လိုမျိုးပေါ့။ အဲ့တာတွေကြောင့် အလုပ်မတွင်တာမျိုးတွေလည်းရှိတတ်တယ်။ အဲ့အချိန်ကျ ဒီ extension သုံးပြီးတော့ site ကို block လုပ်ထားလိုက်တယ်၊ မဖြစ်မနေအချိန်မှ unblock ပြန်လုပ်ပြီးသုံးလိုက်တယ်။ တစ်ကယ်အလုပ်တွင်ပါတယ်။
https://chrome.google.com/webstore/detail/blocksite-stay-focused-co/eiimnmioipafcokbfikbljfdeojpcgbh

JSONView

Json format တွေကို browser မှာ print ထုတ်တဲ့အခါသပ်သပ်ရပ်ရပ်နဲ့ကြည့်မရတာမျိုးတွေအတွက် ဒီ extension သွင်းထားမယ်ဆို Json format တွေကို pretty print format နဲ့ ကြည့်နိုင်မှာဖြစ်ပါတယ်။
https://chrome.google.com/webstore/detail/jsonview/chklaanhfefbnpoihckbnefhakgolnmc

Clear Cache

Web app တွေရေးတိုင်း cache မိနေတဲ့ issue က ခနခနရှိတတ်ပါတယ်၊ အဲ့အချိန်ကျ history ကိုသွား cache clear တွေလုပ်နဲ့အဆင့်တွေများပါတယ်။ ဒီ extension ကိုသွင်းထားပြီးရင် ကိုယ် setting ချထားတဲ့အတိုင်း one click နဲ့ cache clear လုပ်နိုင်မှာပါ။ သိပ်တော့မသုံးဖြစ်ပေမယ့် အသုံးဝင်လို့ထည့်ထားတာပါ။
https://chrome.google.com/webstore/detail/clear-cache/cppjkneekbjaeellbfkmgnhonkkjfpdn

Web Activity Time Tracker

ကိုယ်ရဲ့ browsing activity တွေကို monitor လုပ်ထားပြီး report အနေနဲ့ပြန်ကြည့်နိုင်ပါတယ်။ ဥပမာဒီနေ့ report ကိုကြည့်မယ်ဆို localhost မှာ ၃နာရီ, Facebook မှာ ၃၀မိနစ်, youtube မှာဘယ်လောက် စသည်ဖြင့် ပြန်ကြည့်နိုင်ပါတယ်။ Daily report ရှိသလို date range နဲ့ရော all time နဲ့ရော report ပြန်ကြည့်နိုင်ပါတယ်။ report တွေပြန်ကြည့်ပြီး ကိုယ်ဘာတွေလုပ်ဖြစ်နေလဲဆိုတာပြန်ဆန်းစစ်လို့ရတာပေါ့။ data ကို csv လည်းပြန်ထုတ်လို့ရပါတယ်။

Full Page Screen Capture

Wappalyzer

Site တစ်ခုကို visit လုပ်ပြီးတာနဲ့ အဲ့ဒီ site က ဘာနည်းပညာတွေသုံးထားလည်းဆိုတာသိချင်ရင် အသုံးပြုတာဖြစ်ပါတယ်။ programming language တွေမှမဟုတ်ဘူး တစ်ခြား analytics tools, widgets , server info တွေကအစပြပေးနိုင်ပါတယ်။ သိပ်တော့မသုံးဖြစ်ပေမယ့် သိချင်တဲ့အခါကျ click လှမ်းထောက်ရုံပဲဆိုတော့ သွင်းထားတာပါ။
https://chrome.google.com/webstore/detail/wappalyzer/gppongmhjkpfnbhagpmjfkannfbllamg

ဒီလောက်ပါပဲ၊ ကျနော်က backend ကိုအဓိကထားပြီးရေးတာဆိုတော့ frontend နဲ့ပတ်သတ်တဲ့ extension တွေသွင်းထားတာတော့ သိပ်မရှိပါဘူး၊ ဥပမာ css အတွက်တော့ဘာတို့ပေါ့။

အခုပြခဲ့တဲ့ extension တွေက ကျနော်အရင်က chrome သုံးတုန်းကသွင်းထားတဲ့ extension တွေပါသလို အခု edge ကိုသုံးတဲ့အချိန်ထပ်သွင်းထားတာတွေလဲပါပါမယ်။ ပြောချင်တာကတော့ chrome, edge နှစ်ခုလုံးမှာရနိုင်ပါတယ်။ firefox သုံးတဲ့သူတွေအတွက်တော့ မသေချာပေမယ့် မရှိခဲ့ရင်တောင် alternative extension တွေရှိမယ်ထင်ပါတယ်။
