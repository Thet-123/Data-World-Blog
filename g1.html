Improving your logical thinking skills

Category : General

Programming မှာ ကိုယ့်ကိုကိုယ် Logical thinking skill အရမ်းမကောင်းဘူးလို့ခံစားနေရတဲ့သူတွေအတွက် ဒီ article ကအနည်းနဲ့အများအထောက်အကူပြုနိုင်မယ်လို့ထင်ပါတယ်။

Logical thinking ကောင်းချင်တယ်ဆို ပထမဦးဆုံးအနေနဲ့ စဉ်းစားရမှာကိုမကြောက်ဖို့လိုပါတယ်။ စဉ်းစားရင်းနဲ့ပဲ ပိုပြီးစဉ်းစားတတ်လာအောင်လုပ်ယူနိုင်ဖို့အရေးကြီးပါတယ်။ Programming ဆိုတာကလည်း တစ်နည်းအားဖြင့် Problem solving ပဲဖြစ်တဲ့အတွက် Problem တစ်ခုကိုဖြေရှင်းနိုင်ဖို့ အဆင့်ဆင့်ခွဲချ စဉ်းစားပြီးဖြေရှင်းဖို့လိုအပ်ပါတယ်။ ဥပမာတစ်ခုပေးရရင် ကြက်သားဟင်းတစ်ခွက်ချက်တယ်ဆိုပါစို့၊ ဒီဟင်းဖြစ်လာဖို့ step တွေခွဲချလိုက်ဖို့လိုပါတယ်။ ကြက်သားကိုင်မယ်၊ bite size လေးတွေလှီးထားမယ်၊ အရသာနယ်ပြီးနှပ်ထားမယ်၊ ဒါကကြက်သားအတွက်တစ်ပိုင်း။ ကိုယ့်ဟင်းက အနှစ်နဲ့ချက်မှာဆို curry paste အတွက် ကြက်သွန်နီ၊ဖြူ … etc ဒါတွေကတစ်ပိုင်းထွက်မယ်။ ပြင်ဆင်စရာရှိတာတွေပြင်ဆင်ပြီးပြီဆို execution step ကိုဆက်သွားလို့ရပီ။ ဒါထည့်မယ်၊ ဒါပြီးရင်ဒါထည့်မယ် စသည်ဖြင့်။ Programming မှာလည်းထိုနည်းလည်းကောင်းပဲ။ Problem set A ဆိုတဲ့အရာကိုဖြေရှင်းဖို့အတွက် sub problem တွေစဉ်းစားပြီး ခွဲချလိုက်လို့ရတယ်။ sub problem တွေစဉ်းစားပြီးအဖြေရပြီဆို merge & execution လုပ်ပြီး bigger problem ကိုဖြေရှင်းလို့ရသွားမယ်။

Data structure နဲ့ algorithms ကိုလေ့လာဖို့အကြံပေးချင်ပါတယ်။ CS ကျောင်းသားတွေကတော့ သင်ခဲ့ပေမယ့် ကျနော်တို့လို CS ကျောင်းသားမဟုတ်တဲ့သူတွေကတော့ အချိန်ပေးပြီးလေ့လာသင့်ပါတယ်။ Algorithm ဆိုတာကလည်း Problem တစ်ခုကိုဖြေရှင်းဖို့အတွက်လုပ်ပေးထားတဲ့ steps တွေပဲ။ ဒါကြောင့်ဒါတွေကိုလေ့လာမယ်ဆို ကိုယ့်ရဲ့တွေးတောနိုင်စွမ်း logical thinking skill ကိုလည်းပိုပြီးတော့တိုးတက်လာစေမှာဖြစ်ပါတယ်။

Algorithm လေ့လာပြီးတော့လည်း စစချင်းမှာအရမ်းခက်တဲ့ problem set တွေသွားပြီးတော့ဖြေရှင်းကြည့်လို့မရသေးဘူး။ ကိုယ့်မှာက logical thinking ကို process လုပ်နိုင်ဖို့အတွက် muscle memory ကောင်းကောင်းလည်းမရှိသေးသလို အလေ့အကျင့်လည်းမရှိသေးဘူး။ ဒါကြောင့် problem set အသေးလေးတွေကနေတစ်ဆင့်ခြင်းဆီစဉ်းစားပြီးဖြေရှင်းသွားဖို့လိုတယ်။ ပြီးမှ medium, hard ကိုဆက်သွားပေါ့။ ကိုယ်လုံးဝစဉ်းစားလို့မရတော့ဘူးဆို တစ်ခြားသူတွေဘယ်လိုဖြေရှင်းထားလဲ၊ သူတို့ဘယ်လိုစဉ်းစားသွားလဲဆိုတာကို ကိုယ်ကလိုက်စဉ်းစားပြီးတော့ အလားတူ problem set တွေကိုဖြေရှင်းဖို့ကြိုးစားကြည့်လို့ရပါတယ်။

ဒီလို problem တွေကိုဖြေရှင်းဖို့ကြိုးစားတဲ့နေရာမှာလည်း တန်းပြီး code ရေးတာထက်စာရင် ကိုယ်နားလည်ထားတဲ့ thinking ကို pseudo code နဲ့အရင်ချရေးကြည့်သင့်တယ်၊ ချရေးရင်းနဲ့မှ ဘယ်နေရာကတော့လိုနေတယ်၊ ဘယ်နေရာကိုတော့ထပ်ဖြည့်လို့ရမယ်ဆိုတဲ့အတွေးတွေပိုတွေးနိုင်တယ်။ ကိုယ်လည်းပိုမှတ်မိလာလိမ့်မယ်၊ မှတ်မိလာမယ်ဆိုလို့အလွတ်ကျက်ခိုင်းတဲ့ပုံစံမျိုးလည်းမဟုတ်ဘူး။

ကျနော်အပေါ်မှာ ဥပမာပေးခဲ့တဲ့ ကြက်သားဟင်းဆိုတဲ့ problem set ဆိုပါစို့၊ sub problem အနေနဲ့ ကြက်သားကိုင်ပြီးအရသာနှပ်တဲ့အပိုင်းနဲ့ curry paste လုပ်တဲ့အပိုင်းခွဲလို့ရမယ်။ ပထမဆုံးကြက်သားဟင်းချက်တဲ့အချိန်မှာတော့ sub problem ခွဲတာတွေ sub problem အတွက် algorithms တွေစဉ်းစားတာတွေကကြာကောင်းကြာပေမယ့် ဒုတိယ၊တတိယအကြိမ်ချက်ဖြစ်တဲ့အချိန်မှာတော့ ဒီ algorithm ဟာကိုယ်နဲ့ရင်းနှီးနေပြီးဖြစ်တဲ့အတွက် ရှည်ရှည်ဝေးဝေးစဉ်းစားစရာမလိုဘဲ တန်းပြီးတော့ execution လုပ်နိုင်သွားမှာဖြစ်ပါတယ်။ Programming မှာလည်းထိုနည်းလည်းကောင်းပါပဲ၊ algorithm တစ်ခုကို ကိုယ်ကရင်းနှီးနေပြီးသားဆို Apply လုပ်ရတာပိုမိုလွယ်ကူစေမှာဖြစ်ပါတယ်။

နောက်တစ်ခုက Problem set မတူပေမယ့်လည်း sub problem တွေမှာ ကိုယ်သိထားတဲ့ algorithm ကအသုံးဝင်တယ်ဆိုထည့်သုံးလို့ရပါတယ်။ ခုနကကြက်သားဟင်းမှာထည့်သုံးထားတဲ့ curry paste လုပ်တဲ့ algorithm ကို ဝက်သားဟင်းချက်တဲ့နေရာမှာလည်း curry paste အဖြစ်ပြန်သုံးလို့ရသလိုမျိုးပေါ့။ ဒီလိုမျိုး ဒက်ကနဲသိနိုင်ဖို့ဆိုတာကလည်း အဲ့ဒီ algorithm ရဲ့ အသုံးဝင်ပုံနဲ့ usage ကိုပိုင်နိုင်နေဖို့လိုပါတယ်။ တစ်နည်းအားဖြင့်လေ့ကျင့်မှုရှိဖို့လိုပါတယ်။ ဒါကြောင့်လည်း pseudo code ပုံစံမျိုးချရေးပြီး လေ့ကျင့်တာကောင်းတယ်လို့ပြောခဲ့ခြင်းဖြစ်ပါတယ်။

Internet ပေါ်မှာလည်းတစ်ခြားသော Logical skills ကိုတိုးတက်အောင်လုပ်ပေးနိုင်တဲ့ games လေးတွေ နောက်ပြီးတော့ algorithm သုံးပြီး problem set တွေဖြေရှင်းလို့ရတဲ့ site တွေအများကြီးရှိပါတယ်။ ဒီဟာတွေကနေတစ်ဆင့်လည်းလေ့ကျင့်မှုတွေရယူနိုင်ပါတယ်။

နောက်ဆုံးပိတ်အနေနဲ့ပြောရရင် ဒီ skill ကဒီနေ့လုပ်ဒီနေ့ရတာမျိုးလည်းမဟုတ်သလို Logical thinking skill ဂိတ်ဆုံးရောက်သွားပြီဆိုတာမျိုးလည်းမရှိပါဘူး။ ဒါကြောင့်မို့အချိန်ယူပါ၊ စာများများဖတ်ပါ၊ များများစဉ်းစားပါ၊ စဉ်းစားတဲ့နေရာမှာလည်း ဘောင်အပြင်ကိုထွက်ပြီး စဉ်းစားမယ်ဆိုရလဒ်တွေပိုကောင်းနိုင်ပါတယ်၊ Problems တွေဖြေရှင်းပါ၊ လေ့ကျင့်မှုများများလုပ်ပါ။ ဒီလိုနဲ့ဖြည်းဖြည်းခြင်း တိုးတက်လာမှာပါ။
